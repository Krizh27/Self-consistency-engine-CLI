# Self-Consistency Answer Engine (CLI)

A minimal CLI application that demonstrates the **self-consistency** technique in Generative AI.

The application sends the same user prompt to **three independent LLMs** in parallel:

* GPT-5 (OpenAI)
* Claude Haiku 4.5 (Anthropic)
* Llama 3.1 8B Instant (Groq)

After collecting all three responses, it sends them to **Claude Sonnet 5**, which acts as an impartial evaluator (judge). Instead of selecting one response, the judge analyzes all outputs, combines their strongest points, removes redundancy or inaccuracies, and generates a new synthesized answer.

---

## Models Used

| Role             | Provider  | Model                |
| ---------------- | --------- | -------------------- |
| Initial Response | OpenAI    | GPT-5                |
| Initial Response | Anthropic | Claude Haiku 4.5     |
| Initial Response | Groq      | Llama 3.1 8B Instant |
| Final Judge      | Anthropic | Claude Sonnet 5      |

---

## Self-Consistency Pipeline

```text
                User Question
                      │
        ┌─────────────┼─────────────┐
        ▼             ▼             ▼
     GPT-5      Claude Haiku 4.5    Llama 3.1 8B
      (OpenAI)    (Anthropic)        (Groq)
        │             │             │
        └─────────────┼─────────────┘
                      ▼
          Claude Sonnet 5 (Judge)
                      │
                      ▼
         Final Synthesized Answer
```

## Features

* CLI-based application
* Parallel API calls to multiple LLMs
* Self-consistency answer synthesis
* Individual model responses
* Token usage for each model
* Final synthesized answer generated by the judge model
* Simple and minimal implementation focused on orchestration logic

## Installation

Install the dependencies:

```bash
npm install
```

Create a `.env` file:

```env
OPENAI_API_KEY=your_openai_key
ANTHROPIC_API_KEY=your_anthropic_key
GROQ_API_KEY=your_groq_key
```

---

## Run

```bash
node index.js
```

Enter your question when prompted.

* GPT-5 response

* Final synthesized answer generated by Claude Sonnet 5

## Project Goal

This project was built to understand the orchestration of multiple LLMs and the self-consistency technique by generating independent responses and synthesizing them into a higher-quality final answer using a dedicated judge model.
